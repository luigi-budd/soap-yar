Soap YAR is a mostly finished mod for the game [Sonic Robo Blast 2](https://www.srb2.org/). This mod is only designed to work with SRB2 versions 2.2.15+, and is developed with [SRB2-edit](https://github.com/luigi-budd/SRB2-edit) in mind. The only precompiled builds officially hosted are here on GitHub through the Actions tab.


# Compilation

The easiest method to compile the mod is directly downloading the source, and putting it into a zip file.

1. Obtain the source code
2. Put the `src` folder into a zip
3. Rename the zip's extension to ".pk3"
4. Load the pk3 in SRB2

You can also use SLADE to compile builds, though this method is a bit finnicky on SLADE v3.2+.

https://slade.mancubus.net/

1. Obtain the source code
2. Open the `src` directory in SLADE
3. From the top bar, Archive -> Build Archive
4. Save the pk3 within your SRB2 directory
5. Load the pk3 in SRB2

You can also skip compilation by loading the directory directly in SRB2 with the `addfolder` command.

# Releases

The only releases are in the Actions tab. Simply go to the Actions tab of the repository, click on the latest commit, and download the artifact from the bottom of the page.
*If you a member of the Discord server, the `#gitbuilds` channel has direct downloads of every commit from the `main` branch.*

***Note: There are 2 branches that are both frequently worked on. For everything fun and nonsensical, download artifacts from the `bloat` branch. For the no-nonsense base mod, download artifacts from the `main` branch.***
